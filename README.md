# 🤖 AI-Driven Playwright Automation Framework

This project demonstrates an AI-assisted End-to-End automation framework built using:

- Playwright
- TypeScript
- VS Code
- GitHub
- AI-assisted test generation (Copilot / LLM Agent style prompts)

---

## 🚀 Project Overview

This framework showcases how AI can:

- Convert Gherkin feature files into Playwright tests
- Generate production-ready TypeScript test files
- Follow existing project conventions
- Reuse helper navigation patterns
- Use stable locators

---

## 🏗 Project Structure

tests/ → Playwright test specs  
pages/ → Page Object Models  
features/ → Gherkin feature files  
utils/ → Reusable helpers  
dom/ → DOM locator references  

---

## ▶️ How to Run

Install dependencies:

npm install

Run tests:

npx playwright test

Run in headed mode:

npx playwright test --headed

View report:

npx playwright show-report

---

## 🎯 Key Features

- Multi-browser execution (Chromium, Firefox, WebKit)
- Screenshot & video capture
- AI-style test generation
- Structured test.describe & test.step usage
- Production-ready E2E spec design
